# MIDI4CV

Eurorack module to interface CV modules from MIDI.

# FEATURES

- Outputs available
    - Four analog outputs
    - Four digital outputs

- Modes:
    - Mono
    - Dual
    - Quad
    - Gate

# BUILDING

To compile the project you should provide the path to your GNU toolchain path to the makefile.

```
make all GCC_PATH=<path_toolchain_bin>
```
